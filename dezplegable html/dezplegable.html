Etiquetas y propiedades comunes en CSS

color: para definir el color del texto

font-family: para definir la familia de fuentes del texto

font-size: para definir el tamaño de la fuente

font-weight: para definir el peso de la fuente (normal, bold, etc.)

text-decoration: para decorar el texto (subrayado, tachado, etc.)

background-color: para definir el color de fondo de un elemento

padding: para definir el espacio interior de un elemento

margin: para definir el espacio exterior de un elemento

border: para definir el borde de un elemento

width: para definir el ancho de un elemento

height: para definir la altura de un elemento

display: para controlar cómo se muestra un elemento en la página

position: para controlar la posición de un elemento

float: para controlar la alineación de un elemento dentro de su contenedor

text-align: para alinear el texto dentro de un elemento

list-style: para definir el estilo de viñetas o numeración de una lista

z-index: para definir el orden de apilamiento de elementos superpuestos

opacity: para definir la opacidad de un elemento

box-shadow: para agregar una sombra alrededor de un elemento

border-radius: para agregar esquinas redondeadas a un borde

text-transform: para transformar el texto a mayúsculas, minúsculas o capitalizado

Etiquetas basicas de HTML

body: para el contenido

head: para información sobre el documento

div: división dentro del contenido

a: para enlaces

strong: para poner el texto en negrita

b: para poner el texto en negrita

br: para saltos de línea

H1…H6: para títulos dentro del contenido

img: para añadir imágenes al documento

ol: para listas ordenadas

ul: para listas desordenadas

li: para elementos dentro de la lista

p: para párrafos

span: para estilos de una parte del texto
